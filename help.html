How To Use The Currency Calculator

  • The Basic Currency Conversion
    • First, select the amount of the current currency you wish to convert.
    • Second, select the current currency from the dropdown menu on the left.
    • Third, from the dropdown menu on the right, select the currency you wish to convert the amount to.
    • Fourth, press the button labeled "Convert".
    • Below the conversion entry box two display boxex should appear.
    • The first box contains the currency conversion difference between the original currency and the converted currency.
    • The second box contains the number of google shares that could be purchased with the amount of currency entered
  • Pinned Currency Exchange Rate
    • First, press the button labeled "Pin Currency Exchange Rate" in order to bring up the pin menu.
    • Second, once the pin menu is displayed, pick your first currency by selecting the left drop down menu.
    • Third, select the second currency by selecting the dropdown menu on the right.
    • Fourth, click on the button with the label "Add" to pin the exchange.
    • Lastly, their should be a new display box that gives the rate of the exchange between the two currencies.
